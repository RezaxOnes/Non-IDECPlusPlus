# Command
- rm -r <files or path>
- cd .. = from ../.. to ..
- cd - = return to previous cd 
- edit <Name IDE or Editor> <files or path>
 vm = vim
 vsc = visual studio code
 gte = gnome text editor
Example: edit gte ./test.cpp
- run and debug = run and debug fixed file in command you can edit it
- m-run = manual run
Example: >> ./test.cpp
